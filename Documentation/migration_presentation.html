Project Overview

Strategic Approach

Technical Implementation

Core Components

CPANBridge.pm

  • Base class for all replacements
  • Handles Perl-Python communication
  • JSON serialization/deserialization
  • Error handling and retries
  • Platform-specific execution

cpan_bridge.py

  • Python request router
  • Security validation
  • Dynamic module loading
  • Structured error responses

Module Pattern

Each CPAN Replacement

  • Perl Wrapper: Identical API to original
  • Bridge Layer: JSON communication
  • Python Backend: Actual functionality

Example: DBI Replacement

  • DBIHelper.pm - Perl interface
  • CPANBridge.pm - Communication
  • database.py - Oracle/Informix backend
